# Evaluación de la Aplicación Web FINANZA VIVA

## Descripción

Encuesta realizada después de que los usuarios prueben la aplicación web FINANZA VIVA, con el fin de evaluar la experiencia de uso, funcionalidad, diseño y utilidad del sistema.

## Objetivo

La presente encuesta tiene como objetivo conocer cuánto aprendieron de educación financiera los usuarios, identificando cómo sus conocimientos, hábitos de ahorro y toma de decisiones económicas mejoraron después de utilizar la plataforma.

## Aspectos Evaluados

- Facilidad de uso
- Diseño de la interfaz
- Organización de funciones
- Utilidad de las herramientas financieras
- Experiencia general del usuario
